Reviewing the Graphs

After opening a project, click on the Test Graphs & Charts icon in project sidebar to view any readability tests that were graphically plotted.

A Word Breakdown bar chart is always included with any project. It is basically a graph showing the total counts of each type of word (e.g., 3+ syllable words) and how each category compares to one another. This bar chart will appear like this:

In the above example we can see that there are 1,926 6+ character words, but only 803 3+ syllable words. From this, we could quickly infer that to lower the document's difficulty we should concentrate on replacing words based on their length instead of (syllabic) complexity. Lists of these words can be found by clicking on the Difficult Words icon on the project sidebar.

There are two additional graphs that will be present if you had included the Fry or Raygor graph tests.
