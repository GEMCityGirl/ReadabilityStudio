Project Settings

The options on this tab enable you to customize your project's level of detail and how it connects to the document it is analyzing.

Document linking and embedding

Embed the original document's text in the project file: Select this option to include the document's text directly into the project. This option is useful if you do not want to reanalyze the source document (if it has been altered) the next time you open the project.

Reload the source document when opening project: Select this option to reload and reanalyze the source document the next time you open the project. Note that the source document's content will not be stored in the project, so if the document can't be found then only the results from the previous time the project was opened will be shown.

Statistics display

Minimal: Select this option to only include the most basic information in the statistics and readability windows.

Medium: Select this option to include the most common statistics in the statistics and readability windows.

Verbose: Select this option to include fully detailed information in the statistics and readability windows.
