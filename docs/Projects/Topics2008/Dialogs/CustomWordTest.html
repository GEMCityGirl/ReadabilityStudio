Custom Word Test

This dialog is used to edit or create a custom familiar word test.

Name of the test: If creating a new test, enter its name in this field.

File containing familiar words: Enter the path to the familiar-word list in this field. This list must be a text file where each word is separated by a space, tab, comma, semicolon, or new line. An example file would look like this:

computer install program videogame

or like this:

computer,install,program,videogame

or like this:

computer
install
program
videogame

Familiar words: This list displays all of the words from your familiar-word file.

Associate with document type: Select the document types that you want this test to be associated with when creating a new project. This will only take effect if you choose to have the project wizard select tests for you based on document type.

Associate with industry: Select the industries that you want this test to be associated with when creating a new project. This will only take effect if you choose to have the project wizard select tests for you based on industry or field.

Also include Dale-Chall familiar word list: Select this option to use the Dale-Chall word list in conjunction with your word list when determining if a word is familiar.

Also include Spache familiar word list: Select this option to use the Spache word list in conjunction with your word list when determining if a word is familiar.

Familiar words must appear on all included word lists: Select this option to consider words familiar only if they appear on your custom list and other selected lists. This option is only recommended for special situations where you would only want to find words that appear within a union of your word list and the DC and/or Spache word lists.

Note that this option is only available if you are including the Dale-Chall and/or Spache word lists.

Also treat proper nouns as familiar words: Treats proper nouns found in your documents as familiar words (in conjunction with your word list).

Also treat numerals as familiar words: Treats numeric words found in your documents as familiar words (in conjunction with your word list).

Readability formula: Specifies which readability formula to use when calculating the grade level for this test.

Use stemming to search for similar words: Specifies whether or not to use stemming when comparing your familiar words with the words in your document.

Stemming compares words based on their root and is useful for finding words that might appear in a form different from how they are entered in your familiar word list. For example, let us say that you enter the word "installation" in your familiar word file. If you select English as your stemming type, then occurrences of the words "installation", "install", "installed", "installs", etc. found in your document will be considered familiar. This is because each of these words share the same root, "install". The benefit of this is that you do not need to enter every form of every word into your familiar word file.

If you are not using stemming, then it is recommended that you enter each form of your words (e.g., possessive, plural) in your familiar word list.
