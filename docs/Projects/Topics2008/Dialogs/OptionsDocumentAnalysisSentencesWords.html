Sentences & Words

The options on this tab enable you to specify how your documents should be parsed and analyzed.

Consider sentences overly long if

Longer than 20 words: Select this option to highlight sentences that are 20 words or longer in the "long sentences" window of the document viewing area.

Outside of sentence outlier range: Select this option to highlight sentences that are comparatively longer than other sentences in the current document. These highlighted sentences can be viewed in the "long sentences" window of the document viewing area. This option is only recommended for situations where most of the sentences are an acceptable length, but a few (very) long sentences skew the average. This option is useful for finding these type of sentences.

Sentence deduction

Ignore blank lines when determining the end of a sentence: Select this option to not consider incomplete sentences followed by blank lines as the end of the current sentence and paragraph. If an incomplete sentence is encountered, any blank lines after it will be skipped and the sentence will be merged with the following sentence.

This option is not recommended for documents that contain headers and lists that rely on blank lines to separate them from the narrative text.

Also note that this option is disabled if Always treat an end of line as a new paragraph is selected. In this case, the end of a line will always indicate the end of a sentence and paragraph, whether or not it is followed by blank lines.

Sentences must begin with capitalized words: Select this option to have sentence detection assume that all sentences begin with capitalized words. By default, Readability Studio takes into consideration writing which does not capitalize the first word of every sentence. To change this behavior, select this option and Readability Studio will be more grammatically strict when deducing sentences.

Incomplete Sentences

Exclude from analysis: Select this option to ignore all incomplete sentences when calculating statistics.

Include in analysis and treat as full sentences: Select this option to treat all incomplete sentences as regular sentences when calculating statistics. This is only useful for forms that may contain numerous titles and lists items.

End of paragraph

Only treat an end of line as a new paragraph if following a valid sentence: Select this option to only consider line feeds in the text as the end of the current sentence and paragraph if it follows a valid sentence. Multiple consecutive line feeds (normally indicating a header or beginning of a new chapter) will also be considered the start of a new paragraph.

Always treat an end of line as a new paragraph: Select this option to always consider line feeds in the text as the end of the current sentence and paragraph. This should only be used for special cases where a document does not have any sort of paragraph indenting and contains numerous lists and headers that do not have any blank lines between them.

Numeral syllabication

Treat numeral words as one syllable: Select this option to have the application consider numeric words (e.g., numbers, dates, phone numbers) as monosyllabic.

Sound out each digit: Select this option to have the application sound out each digit when counting the syllables of numeric words (e.g., numbers, dates, phone numbers). For example, the word "555-4778" will be considered nine syllables--read as "five-five-five-four-seven-seven-eight". As another example, "$52" will be considered four syllables--read as "five-two-dollars".
