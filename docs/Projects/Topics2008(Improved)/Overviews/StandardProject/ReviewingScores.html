Reviewing the Test Scores

After opening a project, click on the Readability Scores icon on the project sidebar to view all the test scores, as well as their respective explanations and recommendations.

The top area of the Readability Scores window is the score grid. In here there are four columns:

  • Test: the name of the readability test
  • U.S. Grade Level: the grade level that the test determined the document is suitable for
  • Reader Age: the reader's age that the test determined the document is suitable for
  • Scale Value: the score value for tests that use special scales

Note that most tests have just one type of result (e.g., grade level), so non-applicable columns are left blank. For example, note below that the Gunning Fog result is a U.S. grade level of 9.1. This translates to a ninth grade level reader (in his or her first month of class). Because this test returns a U.S. grade level and age and does not use any a scale, the Scale Value column is empty. Another example is the result for the Flesch Reading Ease test, which is a scale value 0f 80. This test has its own scale, instead of a grade level, so its age and grade level columns are empty.

Select any test in the results grid to display an explanation of its score in the window below.

Along with an explanation of the test's results, tips are also provided for how to lower the score, as well as a summary of what type of text this test is appropriate for.

Finally, the averages (means) of your test scores and ages are displayed at the bottom of the results window. Their respective medians and standard deviations are also included if your statistics level of detail is set to verbose. Note that when calculating these statistics, failed test scores are treated as missing observations and not included. For example, let us say that you included 5 tests, but one of them could not be calculated. In this case, the valid 4 scores will be added and divided by 4 (not 5) to acquire the means.

Some special scores will also be converted when being aggregated into these statistics. The value "college+" found in LIX will be seen as 13 when factored into these statistics. Also, grade ranges such as "5-6" from Dale-Chall will be converted to its own average (e.g., "5-6" will be 5.5).

To remove any test in this list from the project, select it and press the DELETE button.

This window can be printed or exported.
