RIX Test

The Rate Index (RIX) readability formula is useful because it can be used on documents of any Western European language. This test calculates an index score of a document based on sentence count and number of long words (i.e., words containing seven or more characters).

The RIX Formula

I = L/S

Where:

I RIX index score
L Number of long words (7+ characters)
S Number of sentences
