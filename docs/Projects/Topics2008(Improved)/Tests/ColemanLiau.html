Coleman-Liau Test

The Coleman-Liau formula calculates the U.S. grade level of a sample based on sentence lengths and number of characters.

Coleman-Liau is based on text from the .4 to 16.3 grade-level range and best suited for secondary age* reading materials. It usually yields the lowest grade when applied to technical documents.

The Coleman-Liau Formula

First, calculate the estimated Cloze percentage with the following formula:

E = 141.8401 - (.214590*C) + (1.079812*S)

Then convert the Cloze percentage into a U.S. grade level with the following formula:

G = (-27.4004*(E/100)) + 23.06395

Where:

E Estimated Cloze %
G U.S. grade level
C Number of characters
S Number of sentences

Note that a simplified version of this formula is also available, although it yields slightly less accurate precision:

G = ((5.88*C)/W) - ((29.5*S)/W) - 15.8

Where:

G U.S. grade level
W Number of words
C Number of characters
S Number of sentences

This test is designed for 100-word samples, with 3 samples being recommended for best results. Note that Readability Studio analyzes your entire document and does not use subsamples for its test calculations. If a formula requires a subsample of a specific size, then standardization is used.

* 4th grade to college

Citations

Coleman, M. and Liau, T.L. (1975). 'A computer readability formula designed for machine scoring', Journal of Applied Psychology, 60, No. 2, 283-284.
