Revised Spache Test

The Revised Spache test calculates the U.S. grade level of a document based on sentence length and number of unfamiliar words. This test uses a list of common words that are familiar to most younger readers (4th grade and below). Any word in a document that does not appear on this list is considered unfamiliar. Note that all numerals and proper nouns are also considered familiar.

Revised Spache is generally used for primary age* readers to help classify school textbooks and literature. This differs from New Dale-Chall, which is better meant for secondary age readers.

Because this formula uses familiar words (rather than syllable or letter counts), it is often regarded as a more accurate test for younger readers.

Note that Readability Studio uses the revised 1974 version of Spache.

The Revised Spache Formula

G = (.121 * S) +(.082 * U) + .659

Where:

G U.S. grade level
S Average sentence length
U Number of unique unfamiliar words

This test is designed for 100-word samples, with 3 samples being recommended for best results. Note that Readability Studio analyzes your entire document and does not use subsamples for its test calculations. If a formula requires a subsample of a specific size, then standardization is used.

* Kindergarten to 7th grade
