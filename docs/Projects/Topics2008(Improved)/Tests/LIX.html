LIX Test

The Laesbarhedsindex (LIX) readability formula is useful because it can be used on documents of any Western European language. This test calculates an index score of a document based on sentence length and number of long words (i.e., words containing seven or more characters).

The LIX Formula

I = W/S+(100*L)/W

Where:

I LIX index score
W Number of words
L Number of long words (7+ characters)
S Number of sentences

LIX Index Table

0-24 Very easy
25-34 Easy
35-44 Standard
45-54 Difficult
55 and above Very difficult
