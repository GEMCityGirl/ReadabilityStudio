General

The options on this tab enable you to customize the display of the text views in a given project.

Formatting

Document font: Click this button to select the font used in the document viewing area.

Highlighting

Include highlighting legend: Select this option to include a legend of the highlighted items in the text windows.

Highlight text by changing its background color: Select this option to highlight text by changing its background color.

Highlight text by changing its background color: Select this option to highlight text by changing its background color.

Color to highlight difficult words and sentences: Click this button to select the color to highlight words and sentences considered difficult in the document viewing area.

Color to highlight excluded sentences: Click this button to select the color to highlight excluded sentences in the document viewing area. This is only applicable when incomplete sentences are being ignored.

Color to highlight repeated words: Click this button to select the color to highlight repeated words. This highlighting will appear in the Long Sentences & Wordiness document view.

Color to highlight wordy phrases: Click this button to select the color to highlight wordy phrases. This highlighting will appear in the Long Sentences & Wordiness document view.

Paragraph indenting

Indent with 4 spaces: Select this option to indent the paragraphs in the document display window by four spaces.

Use tabs: Select this option to indent the paragraphs in the document display window using a tab.

Do not indent: Select this option to not indent the paragraphs in the document display window.
