Document Views

The options on this tab enable you to add or remove highlighted text views for a given project.

Highlighted document views

Include view of plain text: Select this option to include a window displaying the document without anything highlighted. Note that this document is displayed as the application indexed it. Some formatting may appear different from the original document.

Include view of highlighted hard words: Select this option to include a window that highlights all words containing three or more syllables. Note that this document is displayed as the application indexed it. Some formatting may appear different from the original document.

Include view of highlighted long words: Select this option to include a window that highlights all words containing six or more characters. A character is either a letter or number. Punctuation (e.g., apostrophes) is not part of the character count. Note that this document is displayed as the application indexed it. Some formatting may appear different from the original document.

Include view of highlighted DC unfamiliar words: Select this option to include a window that highlights all words not found on the Dale-Chall familiar word list. Note that this document is displayed as the application indexed it. Some formatting may appear different from the original document.

Include view of highlighted Spache unfamiliar words: Select this option to include a window that highlights all words not found on the Spache familiar word list. Note that this document is displayed as the application indexed it. Some formatting may appear different from the original document.

Include view of highlighted long sentences: Select this option to include a window that highlights all overly-long sentences. Note that this document is displayed as the application indexed it. Some formatting may appear different from the original document.
