General Options and Project Settings

Readability Studio offers many features to customize how documents are parsed and analyzed, as well as how results are displayed. These options can be configured both globally and for individual projects. It is important to note that setting any of the global options will affect all future created projects, but will not affect currently existing projects. To set these options globally, select from the main menu Tools/Options... to open the Options dialog. If you wish to change any options for an individual project, then select from the main menu File/Project Settings... to open the Project Settings dialog.
