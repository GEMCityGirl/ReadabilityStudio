Reviewing the Highlighted Text

After opening a project, click on the View Document icon on the project sidebar to view the text windows that have difficult words (or sentences) highlighted.

The different types of text windows that may be shown (depending on which ones were selected from the Project Wizard) are:

Plain text: This text window is the document with nothing highlighted.

3+ syllable words: This text window contains the document with all of the words that are three or more syllables highlighted.

6+ character words: This text window contains the document with all of the words that are six or more characters long highlighted.

DC unfamiliar words: This text window contains the document with all of the Dale-Chall unfamiliar words highlighted. The Dale-Chall word list is a collection of words that are considered to be familiar to most fourth grade readers. Words in the document that are not found on this list may be less familiar to younger readers and hence more difficult.

Spache unfamiliar words: This text window contains the document with all of the Spache unfamiliar words highlighted. The Spache word list is a collection of words that are considered to be familiar to most fourth grade readers. Words in the document that are not found on this list may be less familiar to younger readers and hence more difficult.

Long sentences: This text window contains the document with all of the overly long sentences highlighted.

To view any of these text windows, simply click on the tab strip:

In the case of the long sentences text window, all overly long sentences will be highlighted.

You can add or remove any of these windows, as well as change both the font and highlighting color, by opening the Project Settings dialog and clicking the Document Display icon. Note that changing these options from the Project Settings dialog will only affect the current project. If you want to make changes to any future projects that you will be making, then go to the Options dialog from the Tools menu instead.

As a special note for documents with numerous sentences over twenty words long, it is recommenced to open the Project Settings dialog and click the Document Analysis icon and select the option Outside of sentence outlier range. This will make is so that only the extremely long sentences (in comparison to the other sentences in the document) are highlighted. This will make it easier to pinpoint the most troublesome sentences that need to be shortened.

Any of these windows can be printed or exported if you want to further review them outside of Readability Studio. To print a text window, go to the File menu and select the Print... option. To save a text window, go to the File menu and select the Export highlighted document... option. You can also find these options on the toolbar or by right clicking on a text window.
