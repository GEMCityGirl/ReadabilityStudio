Reviewing the Test Scores

After opening a project, click on the Readability Scores icon on the project sidebar to view all of the test scores, as well as their respective explanations and recommendations.

The top area of the Readability Scores window is the score grid. In here there are four columns:

  • Test: the name of the readability test
  • U.S. Grade Level: the grade level that the test determined the document is suitable for
  • Reader Age: the reader's age that the test determined the document is suitable for<
  • Scale Value: the score value for tests that use special scales

Note that most tests have just one type of result (e.g., grade level), so non-applicable columns are left blank. For example, note below that the Gunning Fog result is a U.S. grade level of 9.1. This translates to a ninth grade level reader (in his or her first month of class). Because this test returns a U.S. grade level and age and does not use any a scale, the Scale Value column is empty. Another example is the result for the Flesch Reading Ease test, which is a scale value 0f 80. This test has its own scale, instead of a grade level, so its age and grade level columns are empty.

Select any test in the results grid to display an explanation of its score in the window below.

Along with an explanation of the test's results, tips are also provided for how to lower the score, as well as a brief summary of what type of text this test is appropriate for.

Finally, the averages of your test scores and ages are displayed at the bottom of the results window. Their respective medians and standard deviations are also included if your statistics display is set to verbose.

This window can be printed or exported if you want to further review them outside of Readability Studio. To print, go to the File menu and the Print... option. To save, go to the File menu and the Export test scores... option. You can also find these options on the toolbar or by right clicking on this window.
