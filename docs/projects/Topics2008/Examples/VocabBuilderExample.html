Example 2: Improving Students' Vocabulary

In this example, we will save a customized list of the more complex words from a book. If you are an educator, you can send this list to your students and ask them to define these words prior to reading the book. This can be a great way to prepare them for future readings, as well as helping them build their overall vocabulary.

Continuing from example 1, select the top item in the hard word list. Now scroll down to the last word that has six syllables, hold down the SHIFT key, and select this word. Now you will have all of the words with six or more syllables selected. If there are any words that you want to add or remove from your selection, hold down the CTRL key and click on them. Your word list will look like this:

Now that you have selected the words that you want to save, click the Export button on the toolbar:

A Save As dialog will appear and you will have the option to save as either HTML or text. Select where you want to save the file and click Save.

Next, the Export Options dialog will appear. Note that by default, all of the words will be exported. To export just the words that you have highlighted, select the checkbox Only export selected rows. Also, because we are only interested in saving the Word column, not the Syllable Count or Frequency columns, enter 1 in the Columns: to and Columns: from fields. Finally, uncheck the option Include column headers because we are only exporting on column. Your Export Options will look like this:

Click OK to export the word list and then you can distribute this file to your students to help prepare them for their reading assignments.
