Document Views

The options on this tab enable you to add or remove highlighted text views for a given project.

Highlighted document views

Include view of plain text: Select this option to include a window displaying the document without anything highlighted. Note that this document is displayed as the application indexed it. Sentence and paragraph structure may appear different from the original document and some special characters and formatting may also be removed.

Include view of highlighted hard words: Select this option to include a window displaying the document with all words containing three or more syllables highlighted. Note that this document is displayed as the application indexed it. Sentence and paragraph structure may appear different from the original document and some special characters and formatting may also be removed.

Include view of highlighted long words: Select this option to include a window displaying the document with all words containing six or more letters highlighted. Note that this document is displayed as the application indexed it. Sentence and paragraph structure may appear different from the original document and some special characters and formatting may also be removed.

Include view of highlighted DC unfamiliar words: Select this option to include a window displaying the document with all words not found on the Dale-Chall familiar word list highlighted. Note that this document is displayed as the application indexed it. Sentence and paragraph structure may appear different from the original document and some special characters and formatting may also be removed.

Include view of highlighted Spache unfamiliar words: Select this option to include a window displaying the document with all words not found on the Spache familiar word list highlighted. Note that this document is displayed as the application indexed it. Sentence and paragraph structure may appear different from the original document and some special characters and formatting may also be removed.

Include view of highlighted long sentences: Select this option to include a window displaying the document with all overly-long sentences highlighted. Note that this document is displayed as the application indexed it. Sentence and paragraph structure may appear different from the original document and some special characters and formatting may also be removed.
