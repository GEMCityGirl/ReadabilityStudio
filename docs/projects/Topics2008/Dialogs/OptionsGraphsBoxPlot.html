Box Plots

The options on this tab allow you to customize the test score box plots included in batch projects. Note that these options are only available for Readability Studio Professional Edition.

Box Appearance:

Box color: Click this button to select the color used for your boxes.

Solid: Select this option to use a solid color for your boxes.

Use a 3D effect: Select this option to add a 3D look to your boxes.

Apply a color fade across the bar, starting from the bottom: Select this option to apply a color fade across your boxes (starting from the bottom).

Apply a color fade across the bar, starting from the top: Select this option to apply a color fade across your boxes (starting from the top).

Box opacity: Sets the transparency of the box color. Sliding this bar to the far right will set the background to be fully opaque, whereas sliding it to the far left will set the background to be blank.

Box Options:

Display labels on the box and whiskers: Check this option to display labels on the middle points, upper/lower control limits, and whiskers on your box plots.

Box coefficient: Enter the coefficient here to specify the percentiles range used for the upper and lower control limits. For example, a coefficient of 25 will use the 25th and 75th percentiles range (i.e., the quartiles range).
