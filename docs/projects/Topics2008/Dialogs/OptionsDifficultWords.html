Difficult Words

The options on this tab enable you to add or remove difficult word lists for a given project.

Include list of 3+ syllable words: Select this checkbox to include a window displaying a list of all words containing three or more syllables.

Include list of 6+ characters words: Select this checkbox to include a window displaying a list of all words containing six or more characters.

Include list of DC unfamiliar words: Select this checkbox to include a window displaying a list of all words not found on the Dale-Chall familiar word list.

Include list of Spache unfamiliar words: Select this checkbox to include a window displaying a list of all words not found on the Spache familiar word list.
