Export Options

Select Export All from the File menu to display the Export All dialog. This dialog provides numerous options for exporting an entire project into a folder.

Folder to export to: Enter the folder path that you want to export the project to in this field.

Sections to export:

Export text windows: Select this option to export all of the windows from the "View Document" section from the project. This option is only applicable to standard projects that are including highlighted text windows.

Export test scores: Select this option to export all of the windows from the "Readability Scores" section of the project.

Export summary statistics: Select this option to export all of the windows from the "Summary Statistics" section of the project. This option is only applicable to standard projects.

Export graphs: Select this option to export all of the windows from the "Graphs" section of the project. For standard projects, this will include the word breakdown bar chart and the Fry and Raygor graphs (if included in the project). For batch projects, this will include the histograms, box plots, the Fry graph, and the Raygor graph (if included in the project).

Export difficult word lists: Select this option to export all of the windows from the "Difficult Words" section of the project.

Export grammar section: Select this option to export all of the windows from the "Grammar" section of the project.

Export Dolch sight words section: Select this option to export all of the windows from the "Dolch Sight Words" section of the project.

Export Warnings section: Select this option to export all of the warnings of the project. This option is only applicable to batch projects.

Export file types:

Export lists as: Select the file type that you want to save list windows as from this list.

Export text windows as: Select the file type that you want to save text windows as from this list. This option is only applicable to standard projects .

Export graphs as: Select the file type that you want to save graphs as from this list.
