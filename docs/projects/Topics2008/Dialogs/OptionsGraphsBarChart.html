Bar Chart

The options on this tab enable you to customize your bar charts' settings.

Bar Appearance

Solid: Select this option to use a solid color for your bars.

Use a 3D effect: Select this option to add a 3D look to your bars.

Apply a color fade across the bar, starting from the bottom: Select this option to apply a color fade across your bars (starting from the bottom).

Apply a color fade across the bar, starting from the top: Select this option to apply a color fade across your bars (starting from the top).

Bar opacity: Sets the transparency of the bar color. Sliding this bar to the far right will set the background to be fully opaque, whereas sliding it to the far left will set the background to be blank.
