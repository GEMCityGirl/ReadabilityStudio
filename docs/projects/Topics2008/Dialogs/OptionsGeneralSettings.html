General Settings

[This is preliminary Beta documentation]

The options on this tab enable you to manage your user settings.

Settings Management

Import: Click this button to import program settings and defaults from a file.

Export: Click this button to export the current program settings and defaults to a file. This include all of your settings on the Options dialog, your defaults for the New Project Wizard, and your printer headers & footers.

Private Data

Clear: Click this button to clear private data (e.g., most recently used files).
