Dolch Sight Words

[THIS IS PRELIMINARY BETA HELP]

The options on this tab enable you to select which categories of Dolch words to highlight and also how to highlight them.

Highlighting

Color to highlight Dolch pre-primer words: Select this checkbox to highlight pre-primer words in the Dolch text windows, or deselect it to not highlight them. Click the adjacent button to select the color to highlight these words.

Color to highlight Dolch primer words: Select this checkbox to highlight primer words in the Dolch text windows, or deselect it to not highlight them. Click the adjacent button to select the color to highlight these words.

Color to highlight Dolch first grade words: Select this checkbox to highlight 1st grade words in the Dolch text windows, or deselect it to not highlight them. Click the adjacent button to select the color to highlight these words.

Color to highlight Dolch second grade words: Select this checkbox to highlight 2nd grade words in the Dolch text windows, or deselect it to not highlight them. Click the adjacent button to select the color to highlight these words.

Color to highlight Dolch third grade words:: Select this checkbox to highlight 3rd grade words in the Dolch text windows, or deselect it to not highlight them. Click the adjacent button to select the color to highlight these words.

Color to highlight Dolch nouns: Select this checkbox to highlight Dolch nouns in the Dolch text windows, or deselect it to not highlight them. Click the adjacent button to select the color to highlight these words.
