New Fog Count Test

The New Fog Count readability formula is a modified version of the Gunning Fog Index created for the U.S. Navy and was designed for technical documents and manuals. This test calculates the U.S. grade level of a document based on its number of words containing three or more syllables.

This test, along with Simplified ARI and Flesch-Kincaid, is part of the Navy Readability Indices collection of tests.

Note that numerals and proper nouns are always treated as easy words for this test, regardless of their syllable count.

The New Fog Count Formula

G = (((E + (3 * D)) / S ) - 3) / 2

Where:

G U.S. grade level
D Difficult words
E Easy words
S Number of sentences
