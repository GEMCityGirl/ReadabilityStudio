Creating a New Batch Project

To determine the readability of a document collection, you will need to create a new batch project. First, select New Project... from the File menu to open the Select Project Type dialog, as shown below:

Click the Next button to proceed to the Select Files dialog, as shown below:

On this dialog, you can add files from a folder or website to the project.

To add files from a folder, click the Add folder(s) button. You will be presented with the Select Directory dialog:

To search for files from a website and include them in the project, click the Add web page(s) button. This will display the Web Harvester dialog, as shown below:

You can also individually select files to include in the project. Click the Add file(s) button and then select the file(s) that you want to include.

After you have selected your files, you can randomly sample a subset of these documents for your project. Normally, you will want to analyze all the documents that you have selected and ignore this feature. However, some research projects may require random sampling. For these situations, check the Randomly sample option and enter the percentage of documents to analyze.

Note that when the project is first analyzed, the documents that were not included in the sample will be removed from the project.

Click the Next button to continue to the Select Document Structure dialog, as shown below:

In this dialog, you will specify the format that best describes your document. You can select either narrative text, (partially fragmented) narrative text, or non-narrative/fragmented text.

Previews of these document types are shown next to each option. To view the full-size image of any preview, click on it with your mouse.

After selecting your document structure, click the Next button to continue to the Readability Tests dialog, as shown below:

In this dialog, you will select how you want Readability Studio to choose the readability tests to perform. Your first option is to tell Readability Studio which type of document you are analyzing and it will select the most appropriate tests based on that. To do this, select the Recommend tests based on document type option and click the Next button. You will then be presented with the Select Document Type dialog, where you can select which type of document best describes your file (e.g., a literary work or technical form), as shown below:

Your second option is to tell Readability Studio which type of industry your document relates to and it will select the most appropriate tests based on that. To do this, select the Recommend tests based on industry or field option and click the Next button. You will then be presented with the Select Industry dialog, where you can select which field best describes what your file relates to (e.g., magazine publishing or the military), as shown below:

Your final option is to manually select the tests yourself. To do this, select the Manually select tests option and click the Next button. You will then be presented with the Select Readability Tests dialog, where you can select any of the tests that Readability Studio has to offer, as shown below:

Note that you can always add additional tests to your project at any time. If you select either the document type or industry options, you are not bound to just the tests that Readability Studio recommends.

After you have selected the tests to perform, click the Finish button and the project will be created and will appear like this:

From here, you can review the readability scores, box plots, histograms, Fry/Raygor graphs, Dolch statistics, and various lists of difficult words. You may also want to review any warnings that were encountered while analyzing the documents.
