Example 5: Searching for Overly Long Sentences

In this example, we will review a document for overly long sentences. This example will demonstrate the available methods for defining what an overly long sentence. In addition, it will also demonstrate how to search for and review these sentences.

The first step is to open the example file Features. From the Help menu, select the Examples submenu and then select Features.

You will then presented with the Readability Tests dialog, as shown below:

Select the Recommend tests based on document type and click the Next button to proceed to the Select Document Type dialog, as shown below:

Select the Technical document option and click the Next button. The next dialog will be the Select Highlighted Documents dialog, as shown below:

Select all the checkboxes so that we will have all the highlighted text views available. Click the Next button to proceed to the Select Difficult Words Lists dialog, as shown below:

Select all the checkboxes so that we will have all the difficult word lists available and click the Finish button to begin analyzing the project.

If you are excluding incomplete sentences (the default) then this warning message will be displayed.

Because this file contains numerous incomplete sentences, Readability Studio will ask you if excluding them is what you truly intend. In this case, these sentences are just list items that we should ignore, so select No.

Once the project is loaded, the readability scores will be presented. As we can see below, this document scores high within the collegiate range.

This document should be an easy to read list of software features intended for a broad range of readers. These scores are a bit too high, so now we will explore some the document to understand why it is so difficult to read. Select the Statistics icon in the project sidebar to view the document's statistics.

As we can see, there are some very long sentences in this document and is probably the biggest factor in our scores. To view a list of these sentences, click the difficult sentences link. You can also click the Grammar icon in the sidebar and select the Long Sentences tab.

This list can be sorted alphabetically, by word count, or by location. To view any sentence in its original context, double click on it. This will take you to the Long Sentences & Wordiness tab. In this window, all the overly long sentences and wordy phrases are highlighted and each sentence displays its respective word count after it (in parenthesis).

By default, the program considers sentences being overly long if they contain more than 20 words. As we can see, there are a few sentences that are only a few words longer than 20 words that are not so problematic. However, there are a few sentences clustered which exceed 40 words. At this point, we will want to temporarily filter out the smaller of these long sentences and search for the "worst offenders". From the File menu, select Project Settings and then select the Document Analysis icon. As shown below, we have a couple of different ways to determine if a sentence is overly long:

The first option is to define a specific word count as the long sentence boundary. Enter "25" where it says Longer than 20 words and click the OK button. The program will now reanalyze the file, considering that any sentence longer than 25 (instead of 20) are overly long. As shown below, sentences that are between 21-25 words are no longer being classified as overly long:

The second option is to consider sentences overly long if they are comparatively longer than other sentences in the document. From the File menu, select Project Properties and then select the Document Analysis icon. Select the option Outside sentence-length outlier range and click the OK button. At this point, the boundary for overly-long sentences will be calculated based on the outlier range of the sentence lengths. After reanalyzing the file, note how the only sentences highlighted have the lengths of 45 and 48. These are the two "worst offenders" and should be either reworded or split into small sentences.

To see what the outlier range is, click on the Statistics icon in the project sidebar. Here we can see that any sentence more than 43 words is considered overly long.

This option is only recommended for situations where most of the sentences are an acceptable length, but a few (very) long sentences skew the average. This option is useful for finding these type of extreme sentences.
