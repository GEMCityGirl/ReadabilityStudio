## SetBarChartOrientation {-}

Sets the bar charts' orientation.

### Syntax {-}

```{sql}
SetBarChartOrientation(Orientation orientation)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
orientation | The bar charts' [orientation](#orientation).
