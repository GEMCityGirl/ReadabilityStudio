## SetGraphLogoImage {-}

Sets the image to be used for a logo on all graphs.

### Syntax {-}

```{sql}
SetGraphLogoImage(string imageFilePath)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
imageFilePath | The filepath to the image to use for graph logos.
:::
