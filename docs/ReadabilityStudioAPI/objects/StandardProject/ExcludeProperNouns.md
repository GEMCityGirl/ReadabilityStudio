## ExcludeProperNouns {#standard-excludepropernouns .unnumbered}

Specifies whether to exclude proper nouns.

### Syntax {-}

```{sql}
ExcludeProperNouns(boolean exclude)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
exclude | **true** to exclude proper nouns.
:::
