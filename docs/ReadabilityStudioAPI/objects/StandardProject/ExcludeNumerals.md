## ExcludeNumerals {#standard-excludenumerals .unnumbered}

Specifies whether to exclude numerals.

### Syntax {-}

```{sql}
ExcludeNumerals(boolean exclude)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
exclude | **true** to exclude numerals.
:::
