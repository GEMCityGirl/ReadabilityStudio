## SetReviewer {-}

Sets the user name for the software.
This will appear in some output when exporting.

### Syntax {-}

```{sql}
SetReviewer(string name)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
name | The user name for the software.
:::
