## ExportScores {#standard-exportscores .unnumbered}

Saves the project's test scores to *OutputFilePath*.

### Syntax {-}

```{sql}
ExportScores(string outputFilePath)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
outputFilePath | The folder to save the scores.
:::
