## ExportAll {#standard-exportall .unnumbered}

Exports the project's results to the specified folder.

### Syntax {-}

```{sql}
ExportAll(string outputFolder)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
outputFolder | The folder to save the project's results.
:::
