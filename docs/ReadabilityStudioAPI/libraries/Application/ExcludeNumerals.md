## ExcludeNumerals {-}

Specifies whether to exclude numerals from the document analysis.

### Syntax {-}

```{sql}
ExcludeNumerals(boolean exclude)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
exclude | **true** to exclude numerals, **false** to include them.
:::
