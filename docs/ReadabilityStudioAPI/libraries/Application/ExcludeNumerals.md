## ExcludeNumerals {-}

Specifies whether to exclude numerals from the document analysis.

### Syntax {-}

```{sql}
ExcludeNumerals(Exclude)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
Exclude | **boolean** | **true** to exclude numerals, **false** to include them.
:::
