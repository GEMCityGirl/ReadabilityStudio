## SetCenterPrintFooter {-}

Sets the label to appear in the center bottom of printouts.

### Syntax {-}

```{sql}
SetCenterPrintFooter(string label)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
label | The label to appear in the center bottom of printouts.
:::
