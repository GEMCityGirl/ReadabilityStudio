## AggressivelyDeduceLists {-}

Specifies whether to use aggressive list deduction.

```{sql}
AggressivelyDeduceLists(BeAggressive)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
Exclude | Specifies whether to use aggressive list deduction.
