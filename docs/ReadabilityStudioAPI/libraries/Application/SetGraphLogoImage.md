## SetGraphLogoImage {-}

Sets the image to be used for a logo on all graphs.

### Syntax {-}

```{sql}
SetGraphLogoImage(ImageFilePath)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
ImageFilePath | **string** | The filepath to the image to use for graph logos.
:::
