## SetGraphWatermark {-}

Sets the watermark to be written across graphs.

### Syntax {-}

```{sql}
SetGraphWatermark(Label)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
Label | **string** | The text to draw across the graphs.
:::
