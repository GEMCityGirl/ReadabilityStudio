## SetBarChartBarOpacity {-}

Sets the opacity of bar chart bars. **0** is fully transparent, and **255** is opaque.

```{sql}
SetBarChartBarOpacity(OpacityLevel)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
OpacityLevel | The opacity of bar chart bars.
:::
