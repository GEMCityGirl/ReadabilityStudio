## ExcludeProperNouns {-}

Specifies whether to exclude proper nouns from the document analysis.

### Syntax {-}

```{sql}
ExcludeProperNouns(Exclude)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
Exclude | **boolean** | **true** to exclude proper nouns, **false** to include them.
:::
