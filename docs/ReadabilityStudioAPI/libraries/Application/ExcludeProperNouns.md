## ExcludeProperNouns {-}

Specifies whether to exclude proper nouns from the document analysis.

### Syntax {-}

```{sql}
ExcludeProperNouns(boolean exclude)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
exclude | **true** to exclude proper nouns, **false** to include them.
:::
