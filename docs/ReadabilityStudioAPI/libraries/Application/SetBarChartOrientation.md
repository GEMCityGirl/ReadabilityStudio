## SetBarChartOrientation {-}

Sets the bar charts' orientation.

### Syntax {-}

```{sql}
SetBarChartOrientation(Orientation)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
Orientation | **[Orientation](#orientation)** | The bar charts' orientation.
