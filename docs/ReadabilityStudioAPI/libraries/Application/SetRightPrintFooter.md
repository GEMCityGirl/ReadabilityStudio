## SetRightPrintFooter {-}

Sets the label to appear in the bottom right corner of printouts.

### Syntax {-}

```{sql}
SetRightPrintFooter(string label)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
label | The label to appear in the bottom right corner of printouts.
:::
