## SplashScreen {-}

Displays the image from the list of splashscreens at the specified index.

### Syntax {-}
```{sql}
SplashScreen(number index)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
index | Index of which image from the list of splashscreens to display.
:::
