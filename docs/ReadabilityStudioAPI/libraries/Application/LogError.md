## LogError {-}

Logs an error message to be shown once the script is finished running.

### Syntax {-}

```{sql}
LogError(ErrorMessage)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
ErrorMessage | **string** | The error message to log.
:::
