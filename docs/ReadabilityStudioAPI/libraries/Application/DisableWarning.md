## DisableWarning {-}

Disables the specified warning.

### Syntax {-}

```{sql}
DisableWarning()
```

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
WarningId | **string** | The warning's string ID.

Refer to **[EnableWarning()](#enablewarning)** for a list of warning IDs.
