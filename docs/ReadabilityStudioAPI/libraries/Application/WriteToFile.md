## WriteToFile {-}

Writes a string to the specified file path.

```{sql}
WriteToFile(FilePath,Message)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
FilePath | The file to write to (will be overwritten if file exists).
Message | The text to write to the file.
:::
