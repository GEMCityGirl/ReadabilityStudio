## GetFilePathDisplayMode {-}

Returns how filepaths are displayed in batch projects.

### Syntax {-}

```{sql}
FilePathDisplayMode GetFilePathDisplayMode()
```

### Return value {-}

Type: [FilePathDisplayMode](#filepathdisplaymode) \

How filepaths are being displayed.
