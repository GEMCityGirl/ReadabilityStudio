## MsgBox {-}

Displays a message (with optional title).

### Syntax {-}

```{sql}
MsgBox(string message,
       string title)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
message | The message to show.
title | (Optional) The title of the message box.
:::
