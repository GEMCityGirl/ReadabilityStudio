## MsgBox {-}

Displays a message (with optional title).

### Syntax {-}

```{sql}
MsgBox(Message,Title)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
Message | **string** | The message to show.
Title | **string** | (Optional) The title of the message box.
:::
