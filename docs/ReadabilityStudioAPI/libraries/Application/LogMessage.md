## LogMessage {-}

Logs a message to be shown once the script is finished running.

### Syntax {-}

```{sql}
LogMessage(Message)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
Message | **string** | The message to log.
:::
