## ExportSettings {-}

Exports the program's general settings to the specified file.

### Syntax {-}

```{sql}
ExportSettings(SettingsFileToExport)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
SettingsFileToExport | **string** | The filepath to save the program's settings to.
:::
