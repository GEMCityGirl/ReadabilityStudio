## ExportSettings {-}

Exports the program's general settings to the specified file.

### Syntax {-}

```{sql}
ExportSettings(string settingsFileToExport)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
settingsFileToExport | The filepath to save the program's settings to.
:::
