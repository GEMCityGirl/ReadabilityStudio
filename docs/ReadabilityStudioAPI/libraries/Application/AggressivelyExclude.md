## AggressivelyExclude {-}

Specifies whether to use aggressive text exclusion.

### Syntax {-}

```{sql}
AggressivelyExclude(boolean beAggressive)
```

### Parameters {-}

**Parameter** **Description**
| :-- | :-- |
beAggressive | Specifies whether to use aggressive text exclusion.
