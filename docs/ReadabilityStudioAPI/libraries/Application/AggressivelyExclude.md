## AggressivelyExclude {-}

Specifies whether to use aggressive list deduction.

### Syntax {-}

```{sql}
AAggressivelyExclude(BeAggressive)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
Exclude | **boolean** |  Specifies whether to use aggressive list deduction.
