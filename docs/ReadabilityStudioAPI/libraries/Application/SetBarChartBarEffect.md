## SetBarChartBarEffect {-}

Sets the effect used to display bars in bar charts.

```{sql}
SetBarChartBarEffect(BarEffect)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
BarEffect | The [effect](#bareffect) used to display bars in bar charts.
