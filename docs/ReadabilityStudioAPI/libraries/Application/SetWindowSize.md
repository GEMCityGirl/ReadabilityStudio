## SetWindowSize {-}

Resizes the program's window size to the specified width and height.

### Syntax {-}

```{sql}
SetWindowSize(number width,
              number height)
```

### Parameters {-}

**Parameter** | **Description**
| :-- | :-- |
width | The program window width.
height | The program window height.
:::
