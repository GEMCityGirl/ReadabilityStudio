## SetWindowSize {-}

Resizes the program's window size to the specified width and height.

### Syntax {-}

```{sql}
SetWindowSize(Width,Height)
```

### Parameters {-}

**Parameter** | **Type** | **Description**
| :-- | :-- | :-- |
Width | **number** | The program window width.
Height | **number** | The program window height.
:::
