## Clear {-}

Clears\index{Debugger!clearing} the script editor's debug window.

### Syntax {-}

```{sql}
Clear()
```

### Example {-}

```{sql}
-- After running a script, clear any diagnostic info
-- in the editor's debug window.
Debug.Clear()
```
